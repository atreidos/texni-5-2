# Lash Master Telegram Bot

Telegram-бот для записи клиентов к мастеру по наращиванию ресниц.

## Функциональность

- Запись клиентов на услугу (выбор услуги → даты → времени → подтверждение)
- Просмотр своих записей, отмена и перенос
- Автоматические напоминания за 24 и 2 часа до записи
- Управление рабочими слотами мастером через `/addslots`
- Синхронизация с Google Calendar (два календаря: слоты и записи)
- Уведомления мастеру при каждой записи, отмене и переносе

## Стек

- Python 3.11+, aiogram 3.x
- Supabase (PostgreSQL) — основное хранилище
- Google Calendar API — два календаря мастера
- APScheduler — фоновые напоминания
- Деплой: Railway (env-переменные без изменений кода)

## Быстрый старт

```bash
pip install -r requirements.txt
cp .env.example .env   # заполнить токены
python bot.py
```
